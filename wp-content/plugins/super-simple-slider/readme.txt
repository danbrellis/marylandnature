== Description ==

*A lightweight, easy-to-use slider plugin.*

Super Simple Slider's intuitive interface allows you to create great looking, mobile-friendly sliders in minutes with no coding knowledge required. Just choose your images, add some text, drag and drop them into the order you want and you're good to go. The button creation tool gives you the option of adding a button to your slides.

#### Premium Features Include

- **Slideshow Mode** - Set your slider to automatically scroll by enabling slideshow mode
- **Instagram Filters** - Instantly turn your slide images from bland to grand by applying one of the 40 Instragram-style filters
- **Multiple Buttons** - Add up to 3 buttons to any slide and style them just the way you want
- **Transition Effects** - Choose from one of the 8 slide transition effects
- **Navigation Button Style** - Select the style of your navigation buttons
- [And more](https://www.outtheboxthemes.com/wordpress-plugins/super-simple-slider/)

== Installation ==

Installing "Super Simple Slider" can be done either by searching for "Super Simple Slider" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I reorder my slides? =

To change the order of the slides simply drag and drop them into place.

= Do I need to style the text on my slides =

The slide overlay text inherits the styling from the theme that is currently active so there is no styling required.
